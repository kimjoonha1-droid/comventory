# Comventory

재고 및 거래처 관리 시스템

## 프로젝트 소개
중소기업의 재고, 거래처, 제품, 입출고 및 월마감 업무를 관리하기 위한 웹 시스템입니다.

## 기술 스택

- Backend : Java, Spring Boot
- Frontend : JSP, JavaScript
- Database : Oracle DB
- Build Tool : Gradle

## 주요 기능
- 거래처 관리
- 제품 관리
- 입출고 관리
- BOM 관리
- 월마감 관리
- 권한별 접근 제어

## 담당 업무
- 거래처 CRUD
- 거래처 상태 관리
- 권한별 메뉴 제어
- 월마감 기능 구현
